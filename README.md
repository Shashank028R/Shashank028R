* Chat with other users in real time
* Ask questions directly to AI within the room
* Save AI-generated answers for future learners
* Build a shared knowledge repository
* Learn collaboratively with humans and AI

**Example:** In a Mathematics room, a student can ask an AI question, and the answer remains available for everyone in that room.
